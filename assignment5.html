DATA Definition language

Data Definition Language (DDL) is an integral part of SQL (Structured Query Language), specifically designed for defining and managing the schema of a database. Unlike Data Manipulation Language (DML), which focuses on data operations such as insertion, updating, and deletion, DDL is concerned with the structure and organization of the database objects themselves. DDL commands are vital for creating, modifying, and removing database structures, ensuring the database is organized and ready for data manipulation.

  1. CREATE

  2. ALTER

  3. DROP

  4. TRUNCATE

  5. RENAME

  6. COMMENT
